# Agent Operating Standard

This repo uses Guardian Compound Engineering.

## Loop

Plan → Work → Simplify → Review → Compound → Repeat

## Defaults

- Consent, legality, privacy, non-harm, and human flourishing guide agent work.
- Do not paste/store secrets, full SSNs, bank logins, or 2FA codes.
- External sends/posts/DMs require human approval unless explicitly configured.
- Customer-facing copy must pass Stop Slop review before delivery.

## Compound folders

- `docs/brainstorms/` — requirements discovery
- `docs/plans/` — implementation plans
- `docs/solutions/` — solved-problem notes and reusable patterns
- `todos/` — review findings and follow-ups

A meaningful change is not done until it leaves behind reusable learning, validation, or a checklist.

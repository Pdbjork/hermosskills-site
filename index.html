Our editorial policy

Process

From submission to catalog

Every skill passes through a dual-review pipeline before it reaches the catalog. No exceptions.

01

Submit your skill

Package as a SKILL.md file and submit through the catalog portal. Include usage examples, permissions scope, and runtime requirements.

02

Dual review

A human editor checks for factual accuracy, safety, and disclosure compliance. An LLM runs automated red-teaming for prompt injection and output manipulation.

03

Published & tracked

Approved skills enter the catalog with editorial notes. Sponsor relationships are disclosed inline. Security updates trigger catalog bumps with diff notes.
